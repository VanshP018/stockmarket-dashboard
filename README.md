# Stock Market Analysis Dashboard

A comprehensive Tableau-based interactive dashboard for analyzing stock market trends and performance metrics across multiple companies.

## Overview

This project provides detailed analysis and visualization of stock market data for six major tech companies:
- **Apple**
- **Facebook**
- **Google**
- **Nvidia**
- **Tesla**
- **Twitter**

## Dashboard Features

The interactive dashboard includes multiple analysis views:

- **Overview**: High-level stock performance summary
- **Individual Stock Analysis**: Detailed analysis for Facebook, Tesla, and Twitter
- **Moving Average & Open Price Insights**: Trend analysis and price movements
- **End Date and Start Date Comparison**: Temporal price comparisons
- **Highlighted Companies**: Featured company metrics and performance

## Data

Each CSV file contains historical stock data including:
- Opening prices
- Closing prices
- Highest and lowest prices
- Trading volume
- Date information

## Requirements

To view and interact with this dashboard, you need:
- **Tableau Public** (free) or **Tableau Desktop** (commercial)
- Web browser for Tableau Public access

## Usage

1. Download the `StockMarketAnalysis_Dashboard.twbx` file
2. Open it with Tableau Public or Tableau Desktop
3. Interact with filters and visualizations to explore stock trends
4. Use date ranges and company selectors to customize views

## Key Insights

This dashboard enables analysis of:
- Stock price trends over time
- Comparative performance between companies
- Moving average calculations for trend identification
- Price volatility and trading patterns
- Seasonal and temporal trends
